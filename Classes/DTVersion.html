DTVersion Class Reference

Inherits from NSObject
Declared in DTVersion.h

Overview

Class that represents a version number comprised of major, minor and maintenance number separarated by dots. For example “1.2.2”. This encapsulation simplifies comparing versions against each other. Sub-numbers that are omitted on creating a DTVersion are assumed to be 0.
